Επιπλέον βιβλιοθήκες που χρησιμοποιήσατε εκτός της βασικής βιβλιοθήκης της Java.
Κάθε εγγραφή πρέπει να περιέχει τα εξής:
  - Όνομα της βιβλιοθήκης
  - Κλάσεις που χρησιμοποιήθηκαν από τη βιβλιοθήκη
  - path του .jar αρχείου μέσα στον κατάλογο αρχείων που υποβάλλετε
	
	
	-Επιπλέον βιβλιοθήκες για τον έλεγχο του αρχείου xml:
        org.w3c.dom.Element;
        org.xml.sax.SAXException;

	-Κλάσεις των παραπάνω βιβλιοθηκών
        Document, NodeList, Element, SAXException

	-Το εκτελέσιμο αρχείο βρίσκεται στο:
        HW3_Poulianou_Chaido_2613/ce326.hw3/dist/ce326.hw3.jar

Πως μεταγλωττίζεται και εκτελείται ο κώδικας σας. Η μεταγλώττιση και η εκτέλεση πρέπει 
να γίνεται μέσω γραμμής της εντολών (command line). Οι επιλογές που έχετε είναι οι εξής:
  - ant script (προτιμητέο)
  - maven script
  - gradle script
  - δικό σας script

  Σε όλες τις παραπάνω περιπτώσεις παραθέστε αναλυτικά τις εντολές μεταγλώττισης και εκτέλεσης του κώδικα σας.

-Με ant script
Στο φάκελο που βρίσκεται το αρχείο build.xml, δηλαδή στο path HW3_Poulianou_Chaido_2613/ce326.hw3/
Μεταγλώτιση γράφοντας ant compile
Δημιουργία jar αρχείου γράφοντας ant jar
Εκτελεση γραφοντας ant run
ή και χωρίς ant γρλαφοντας 
java -jar "HW3_Poulianou_Chaido_2613/ce326.hw3/dist/ce326.hw3.jar"
